- Important way the businesses respond to and meet customer needs. - Provide personalized, intuitive customer care - Features - Intents - Entities - Training - Integrations - Analytics - Fulfillment
- Chatbot is an application that simulates human conversation, either aloud or via text message - Through typing or talking, a chatbot can connect with a customer. It can influence a customer relationship. - Bots can answer the same straightforward questions over and over again, leaving customer service reps more time to help customers with more complex questions or problems. - Simple items like: - Password reset requests, order status, and store locations and hours. - Consequently, fewer cases get logged for support agents to resolve. - Customers spend less time waiting “in the queue.” ![AI bots](https://res.cloudinary.com/hy4kyit2a/f_auto,fl_lossy,q_70/learn/modules/service_bots_basics/learn-about-einstein-bots/images/5e7739dd71b964a1f319698d192e50ab_service_cloud_setup_process_ai.png)
- NLU (natural language understanding) is the technology that makes Salesforce Einstein, well, smart. - Salesforce Einstein is an NLU technology that trains chatbots to create a learning model. - The learning model helps chatbots created with Salesforce understand customer interactions in a chat window.
- Dialog - Dialogs are conversation snippets that control what a bot can do. Each dialog includes a dialog intent, which is optionally available for NLU training to understand different kinds of customer responses. - Bot moves between several different dialogs. Each dialog handles a portion of the conversation. - For example, Welcome, Main Menu, Order Status, Location and Hours, and Transfer to Agent are individual dialogs that a customer might experience as part of a single conversation with a bot. - Dialog Intent - Actions and generally start with verbs, such as buy, return, find, look up, cancel, edit, etc. - Entity - Entities are a type of data that you want to collect from a customer. - Salesforce provides the following system entities: Text, DateTime, Date, Money, Number, Person, Location, Organization, Percent, Boolean, and Object (standard Salesforce or custom). - ![entity](https://res.cloudinary.com/hy4kyit2a/f_auto,fl_lossy,q_70/learn/projects/build-an-einstein-bot/write-bot-conversations-using-slots-and-entities/images/7c4566f7c8938267433cf20ba5534c93_new_entity.png) - Variable - Container that stores a specific piece of data collected from a customer - You must associate each variable with an entity. - ![var](https://res.cloudinary.com/hy4kyit2a/f_auto,fl_lossy,q_70/learn/projects/build-an-einstein-bot/write-bot-conversations-using-slots-and-entities/images/62632dcb88574b0f041a0b60061080f1_bot-asks.png) - ![var2](https://res.cloudinary.com/hy4kyit2a/f_auto,fl_lossy,q_70/learn/projects/build-an-einstein-bot/write-bot-conversations-using-slots-and-entities/images/1e4c083309d17d7a1f306c05ceb0d199_bot-says.png)
