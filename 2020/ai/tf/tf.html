- TensorFlow is a machine learning system that operates at large scale and in heterogeneous environments - TensorFlow uses dataflow graphs to represent computation, shared state, and the operations that mutate that state. - It maps the nodes of a dataflow graph across many machines in a cluster - within a machine across multiple computational devices, including multicore CPUs, generalpurpose GPUs, and custom-designed ASICs known as Tensor Processing Units (TPUs). ![tf-1](img/tf-1.png)
