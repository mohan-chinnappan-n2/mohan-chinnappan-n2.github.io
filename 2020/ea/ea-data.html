- You can create a remote connection using the Amazon S3 connector to sync CSV data from an S3 bucket to Analytics. ![s3 connector](https://resources.help.salesforce.com/images/8476657213d253a06fecb8dde60a8c62.png)
- The Salesforce Einstein Analytics Connector for Excel gives you a fast, easy way to import data from Excel 2013 into Analytics ![Salesforce Wave Connector for Excel](https://store-images.s-microsoft.com/image/apps.23560.d9d09e8c-b30c-4bd0-92e2-e5c4798974f4.fe47c98a-6367-4ffa-8967-bf7487999492.7a431400-8541-407c-a592-a9f2d55a3e92.png)
- You can use the External Data API to create a single dataset based on external data in the .csv format. You can also use the API to edit the dataset by uploading a new .csv file. - When you edit the dataset, you can choose to overwrite all records, append records, update records, or delete records. ![External API](img/external-data-api.png)
