- WebRTC is an open source project to enable realtime communication of audio, video and data in Web and native apps. - RTCPeerConnection for communicate streaming data between browsers - Signaling: coordinate communication and to send control messages - e.g. Socket.IO - Designed to work peer-to-peer, so users can connect by the **most direct route possible** - Client applications need to traverse NAT gateways and firewalls, - Peer to peer networking needs **fallbacks** in case direct connection fails. - use STUN servers to get the IP address of your computer - TURN servers to function as **relay servers** in case peer-to-peer communication fails. - Encryption is mandatory for all WebRTC components - JavaScript APIs can only be used from secure origins (HTTPS or localhost). ![Exchange](https://hacks.mozilla.org/wp-content/uploads/2013/07/webrtc-complete-diagram.png)
